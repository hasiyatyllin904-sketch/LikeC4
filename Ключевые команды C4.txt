npx.cmd likec4 start  - Запускает локальный сервер с "горячей" перезагрузкой.

npx likec4 export png - Экспортирует схему в PNG например, в папку assets

npx likec4 build - Генерирует статический HTML-сайт с полной интерактивной архитектурой.

likec4 format - Автоматически форматирует ваш .c4 файл (как Prettier для JS).
